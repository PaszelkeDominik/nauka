Lessons wrap-up

Jest GIT!

Git - system kontroli wersji. Dzięki niemu możemy zapisywać jakie zmiany dokonaliśmy w projekcie.
Komendy na start:
git config –global user.name „Jak Kowalski”
git config –globar user.email jankowalski@example.com
Wszystkie pliki z projektu trzymamy w REPOZYTORIUM:
git init
.git
Commit = stan projektu i opis zmian od ostatniego commitu.
Komendy Które trzeba zapamiętać:
git status
git add .
git commit -m „opis komitu/zmian”
git / git lens fajne rozszerzenia do vs code
gitk pokazuje historie komitów w programie graficznym
