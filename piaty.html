WTF: Klan

DOM- A w nim rodzina elementów. Pojedyńczy element to węzeł/node. Relacje między elementami: rodzic-dziecko. Dziedziczenie - dzieci dostają atrybuty rodzica o ile same nie mają go zdefiniowanego. do dziedziczenia używamy własności: inherit;.

JS: funkcje

  • function calculate(myNumber){
    • return myNumber*7;
  • }
Funkcja zawsze coś zwraca. Nawet jak brakuje słowa "return" - wted zwraca undefined.
DRY - Don't Repeat yourself
(Fat)arrow function:
  • const calculate = (myNumber) =>{
    • return myNumber*7;
  • }
lub (skrócony zapis)
  • const calculate = (myNumber) =>myNumber*7;
(Fat)arrow function nie tworzy nowego kontekstu this, nie ma dostępu do arguments i super.
