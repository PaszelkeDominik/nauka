JS: Immutability w obiektach

Własności obiektów można mutować. Zmieniając wartość w obiekcie zmieniamy ją we wszystkich własnościach, które wskazują na ten obiekt. Żeby tego uniknąć, musimy za pomocą dot notation wskazać na własność tego obiektu, tzn. stworzyć nowy obiekt.

Flexbox

display: flex;
-"nowy" system pozycjonowania i rozmieszczania elementów. flex pracuje w dwóch osiach: wiersz/kolumna.
flex-direction:
- właściwość ustawiająca elementy flexu wzglendemjego kontenera. Okteślając główną oś i kierunek.
  • : row; - wartość domyslna
  • : column;
  • : row-reverse;
  • : column-reverse;
flex-wrap:
- wrap- zawijanie wiersza/kolumny. Jeżeli kontent, niemieści się w jednym wierszu/ kolumnie, pozwala na wyświetlenie go w kilku.
  • : nowrap; - wartość domyslna
  • : wrap;
  • : wrap-reverse;
flex-flow:
- shorthand, zawiera dwie własności w jedną.
  • <flex-direction> <flex-wrap>;
justify-content:
- ustawia elementy wzdłuż osi flex-direction. Określa sposób w jaki elementy, sa ustawione wzglendem siebie i/lub ich kontenera.
  • : flex-start; - wartość domyslna
  • : flex-end;
  • : center;
  • : space-between;
  • : space-around;
align-content:
- ustawia rozmieszczenie przestrzeni pomiędzy i dookoła zawartości względem osi przeciwnej do flex-direction.
  • : stretch;
  • : flrx-start;
  • : flex-end;
  • : center;
  • : space-between;
  • : space-around;
align-items:
- ustawia elementy wzdłuż osi przeciwnej do flex-direction.
  • : stretch;
  • : flrx-start;
  • : flex-end;
  • : center;
  • : baseline;
align-self:
- ustawia element wzdłuż osi przeciwnej do flex-direction. Ustawia się bezpośrednio na elemencie.
  • : stretch;
  • : flrx-start;
  • : flex-end;
  • : center;
  • : baseline;
order: 0;
- definiuje kolejność elementu, Wartościa są cyfry. Pierwszeństwo otrzymuje mniejsza liczba. Dopuszcza liczby ujemne.
flex-grow: 0;
- określa ile miejsca, element powinien zajmować, proporcjonalnie do innych elementów.
flex-shrink: 1;
- określa proporcję w jakich elementy się kurczą, jeżeli nie mieszczą się w kontenerze.
flex-basis: __px;
- własność ustawiająca wielkość elementu. Określa rozmiar pola zawartości, o ile nie ustawiono inaczej z wielkością box-sizing.
flex:
- shorthand, zawiera trzy własności w jedną.
  • <flex-grow> <flex-shrink> <flex-basis>;

Normalizacja vs reset

Przeglądarka nadaje własne style. Dlatego warto użyć resetu lub normalizacji. Reset wyłącza domyślne(za pomocą * - selektor uniwersalny, wybiera wszystkie elementy na stronie) ustawienia. Normalizacja, za pomocą osobnego pliku normalize.css ustawiamy własne początkowe style dla nowego projektu. Plik ten pobinien być załadowany przed plikiem z właściwymi stylami.

    Reset:
    *{
    • margin: 0;
    • padding: 0;
    • font-size: 16px;
    • font-weight: normal;
    • outline: none; Uwaga:nie używać!
    }
    ul, li{
    • list-style-type:none;
    }
    Normalizacja:
    html{
    • bos-sizing: border-box;
    }
    *,*:before,*:after{
    • bos-sizing: inherit;
    }

Tła

background-image: url(`ścieżka-do-pliku`);
- ścieżka względem pliku .css
background-position: center center;
- ustawia początkowż pozycję dla tła-obrazu
background-repeat:
- właściwość, która ustawia jak obrazkowe tło się powtarza. Może powtarzać się, horyzontalnie, vertykalnie, na całej powierzchni oraz wcale się nie powtarzać. Za pomocą dwóch wartości (horyzontalna vertykalna) mixujemy efekt, np.background-repeat: space repeat;.
  • : repeat;
  • : repeat-x
  • : repeat-y
  • : round;
  • : space;
  • : no-repeat;
background-attachment:
- właściwość, która ustawia sposób w jaki tło w viewport jest przymocowane(attached).
  • : scroll; -domyślnie
  • : fixed;
  • : local;
background-origin:
- Właściwość nadająca pozycję od, której tło powinno, się zaczynać. From the border start, inside the border, or inside the padding.
  • : border-box;
  • : padding-box;
  • : content-box;
background-clip:
- Właściwość, która ustawia tło elementu w określonej strefie border-box-u.
  • : border-box;
  • : padding-box;
  • : content-box;
  • : text;
background-color:
- kolor. rgb(a) -a- alfa, tzn. przezroczystość tła.
background-size:
- rozmiar.
  • : auto;
  • : cover;
  • : contain;
  • : <width> <height>
